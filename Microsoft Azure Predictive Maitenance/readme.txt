The AZURE PREDICTIVE MAINTENACE dataset was provided by a Kaggle user in the Kaggle plateform

This dataset was available as a part of Azure AI Notebooks for Predictive Maintenance. But as of 15th Oct, 2020 the notebook (link) is no longer available.. 

It consists of hourly average of voltage, rotation, pressure, and vibration collected from 100 machines for the year 2015.

Data Set Link: https://www.kaggle.com/datasets/arnabbiswas1/microsoft-azure-predictive-maintenance
